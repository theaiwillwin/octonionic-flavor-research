# Load-Bearing Associator Benchmark Audit

## Supported claim

A 31-parameter octonion `AssociatorEnergy` model beats the 31-parameter non-octonion `StiefelEnergy` control when the target is generated by explicit multi-slot octonion associator-energy contractions; the control reverses the result on a non-octonion Stiefel teacher.

## Why this benchmark is load-bearing

The target depends on products of full associator vectors `[z,u_i,u_j] = (z u_i)u_j - z(u_i u_j)`, not just an induced 3x3 matrix. The matched Stiefel control keeps the same 7D/3D frame and readout parameter count but removes octonion multiplication and non-associativity.

## Interpretation

- `golden`: AssociatorEnergy wins the associator-family teacher decisively.
- `random_associator`: AssociatorEnergy wins the randomized associator-family teacher even more strongly.
- `stiefel_non_octonion`: StiefelEnergy wins the negative control, which is the correct falsification boundary.

## What is not claimed

- This does not prove octonion layers are universally better than ordinary networks.
- This does not yet test real-world data or dynamical field configurations.
- This isolates a synthetic structural inductive bias: when the generating rule is genuinely non-associative, the matching non-associative model generalizes better than the matched Stiefel surrogate.
