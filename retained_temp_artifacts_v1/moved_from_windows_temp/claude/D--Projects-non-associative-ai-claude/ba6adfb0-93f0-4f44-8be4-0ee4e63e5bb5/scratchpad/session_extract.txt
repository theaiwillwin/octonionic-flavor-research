This was also **not a kill shot**.

The parameter-starved associator matched the unconstrained baseline exactly on test MSE despite having fewer parameters. That means the constrained octonion feature map is not obviously hurting performance on this simple task.

But the decisive result is:

> This particular synthetic benchmark is too linearly degenerate to prove utility.

The target was:

\[
Y = X_{\text{rotated}} \cdot (1,\phi^{-1},\phi^{-2}) + \epsilon
\]

That is an affine/linear regression problem. All three architectures can collapse to an effective affine map:

- `FreeMixerModel`: `Linear(3,9) -> Linear(9,1)` collapses to one affine `3 -> 1` map.
- `AssociatorModel`: row-scaled associator matrix features plus a free predictor also collapse to one affine `3 -> 1` map.
- `Associator+Reg`: same, with the internal ladder forced to φ, but the predictor can absorb the scaling.

So the benchmark mostly measured whether each model can solve noisy linear regression. All can. They all hit the same noise floor.

## What we learned

1. **The strict ablation implementation works.**
   The associator model is parameter-starved and trainable.

2. **Golden regularization works.**
   The internal SVD ladder locks to:

   \[
   (1,\phi^{-1},\phi^{-2})
   \]

   without damaging MSE.

3. **MSE alone does not force φ.**
   The unregularized associator learned useful linear prediction while leaving its ladder around approximately `0.56–0.60 / 0.40–0.44`.

4. **The proposed synthetic hierarchy benchmark is underpowered.**
   Because the target is linear, the free predictor can absorb the ladder geometry. The test does not force the network to use the non-associative structure.

## Bottom line

Phase 2 result so far:

> The associator layer survives the strict ablation but does not yet demonstrate superior utility.

The real next benchmark needs to prevent the predictor from absorbing the geometry. The next version should make the target depend on **pairwise or triadic interactions generated by the associator matrix itself**, not just a linear golden-weighted sum. That would test whether the octonionic constraint gives an advantage where non-associative structure is actually load-bearing.

## New benchmark design

The target is no longer just:

\[
y = x \cdot (1,\phi^{-1},\phi^{-2})
\]

That was too easy.

The new benchmark generates data from a hidden octonionic teacher:

\[
M^\*_{ij} = \|[u_i, u_j, e_7]\|^2
\]

using the locked near-golden frame from `fn_ladder_scan_results.json`.

Labels are generated by:

\[
y = \tanh(\mathrm{flatten}(x_i M^\*_{ij})) \cdot w + b + \epsilon
\]

So the associator matrix is now **load-bearing**. If you permute or destroy the teacher matrix, the labels change.

Models:

```text
Free Mixer:      46 params
Associator:      31 params
Associator+Reg:  34 params
```

The baseline has more parameters and can represent the teacher in principle, but it has to learn a free `3 -> 9` nonlinear feature map from small noisy data. The associator model is constrained to the octonionic teacher class.

## Result: positive utility signal

Command:

```bash
uv run python examples/associator_teacher_ablation.py \
  --epochs 1000 \
  --n-train 48 \
  --n-test 2048 \
  --noise 0.15 \
  --lambda-reg 0.5 \
  --seeds 0 1 2 3 4
```

## Interpretation

This is the first real **Proof of Utility** result.

The free mixer wins on training MSE:

```text

```

But it loses badly on test MSE:

```text

Associator test MSE ≈ 0.025
```

Since the noise level is:

\[
\sigma = 0.15,\quad \sigma^2 = 0.0225
\]

the associator models are close to the noise floor, while the free mixer is clearly overfitting the 48 noisy training examples.

That is exactly the desired behavior:

> The unconstrained baseline memorizes better; the octonion-constrained model generalizes better.

Across the two 5-seed blocks:

```text
Free Mixer average test MSE:      ~0.0553
Associator average test MSE:      ~0.0253
Associator+Reg average test MSE:  ~0.0253
```

So the associator cuts held-out error by roughly half on this teacher-class task.

## What happened to the golden regularizer?

The regularized model locked its ladder near golden ratio, as expected:

The unregularized associator did **not** need to stay golden to generalize well. It often found other valid associator matrices with strong test performance:

That is important. It means there are two separate effects:

1. **Octonion associator constraint** gives useful inductive bias and generalization.
2. **Golden-ratio prior** forces a specific internal hierarchy, but is not required for this teacher task.

## Bottom line

The first synthetic benchmark was inconclusive because it collapsed to linear regression.

The new nonlinear associator-teacher benchmark is positive:

> A parameter-starved octonion associator model generalizes about 2x better than a higher-parameter unconstrained free mixer when the hidden data-generating process is actually associator-structured.

That is not yet proof of general usefulness on real-world data, but it is a real utility result in the correct synthetic regime. The next step should be a sweep over:

- `n_train`: 16, 32, 48, 96, 192
- `noise`: 0.0, 0.05, 0.15, 0.30

That will tell us whether this is specifically “teacher-class recovery” or a broader regularization advantage.
